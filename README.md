# azureappservicedemo

Додайте до проєкту, з яким працювали під час заняття, картинки для товарів.
Користувач сайту має змогу при створенні товару додавати картинку. На сторінці з відображенням товарів також мають відображатись і картинки.
Самі картинки ви маєте зберігати у контейнері ресурсу Azure Storage Account.
